AWTools Parent POMs

At the moment i use these two POMs for my projects. The first POM defines some some common build settings (like file encoding, jdk version, compiler settings, code coverage, my preferred license, some remote repositories and all the plugin versions to build and report). The second POM has the first POM as parent. The second POM defines dependencies (log4j, commons-lang, commons-io, junit, mockito).

POM without lib dependencies (Master POM)

Defines my basic Maven setup (JDK version, repositories, build reports, etc.) For more details have a look at the pom.xml.

POM with lib dependencies (AWTools POM)

This is a branch of the master POM with some dependencies (commons-lang, commons-io, JUnit and Mockito).

Betoffice Core

Betoffice-Storage

Defines database schema, Java entities to accesss the database and a service to read, write, update the database.

Betoffice-Exchange

Creates text files containing all the data of betoffice.

Betoffice-Openligadb

Implements a web service to access the OpenLigaDB to get the newest football results.

Betoffice-Batch

Defines a batch interface to access the betoffice services.

Betoffice-Testutils

Defines some utilities for testing. Used by all betoffice projects.

Betoffice-Test

Integration test for betoffice. This project needs more love.

Betoffice Client

Betoffice-Swing

A fat client for betoffice. This project is not under development. The web frontend will be the future.

Betoffice Web

Betoffice-jweb

The betoffice webservice. Rest like implementation with Spring MVC running on a Tomcat server.

Betoffice-hp

The homepage of betoffice/tippdiekistebier. Based on a XML to HTML transformation.

Betoffice-js

The Javascript API for the betoffice webservice.

Betoffice-jadmin

The betoffice webservice for the administrator. This is an Rest-like implementation with Spring MVC running on a Tomcat server.

Betoffice-webadmin

The homepage for the betoffice administrator. This project uses Bootstrap and AngularJS.
